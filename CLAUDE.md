# LEGACY — 선수 서사 굿즈 플랫폼

## 프로젝트 개요
선수들의 인생 서사(스토리)를 기반으로 한 굿즈 판매 플랫폼.
단순한 굿즈 판매를 넘어, 선수 한 명 한 명의 감동적인 서사와 연결된 경험을 제공한다.

## 파일 구조
```
claude12/
├── index.html     ← 메인 플랫폼 (단일 파일, 모든 CSS/JS 포함)
└── CLAUDE.md      ← 프로젝트 문서 (이 파일)
```

## ✏️ 선수 데이터 수정 방법

`index.html` 파일을 열고, 상단의 `PLAYERS` 배열을 수정하세요.

### 선수 정보 필드 설명

| 필드 | 설명 | 예시 |
|------|------|------|
| `name` | 선수 이름 | `"홍길동"` |
| `number` | 등번호 | `"10"` |
| `position` | 포지션 | `"미드필더"` |
| `team` | 소속팀 | `"서울 FC"` |
| `photo` | 사진 URL (null이면 플레이스홀더) | `"images/player1.jpg"` or `null` |
| `color` | 선수 대표색 (HEX) | `"#FF3300"` |
| `badge` | 카드 뱃지 텍스트 | `"에이스"`, `"캡틴"` |
| `quote` | 선수 대표 명언 | `"포기란 없다."` |
| `excerpt` | 카드 서사 요약 (2-3줄) | `"부상을 딛고..."` |
| `story` | 전체 서사 (모달에 표시) | 긴 텍스트, `\n\n`으로 단락 구분 |

### 사진 교체 방법

1. **로컬 이미지**: `claude12/images/` 폴더에 이미지를 넣고 `photo: "images/player1.jpg"` 로 설정
2. **URL 이미지**: `photo: "https://example.com/player.jpg"` 로 설정
3. **플레이스홀더 유지**: `photo: null` 그대로 두기

### 굿즈 추가 방법

각 선수의 `goods` 배열에 항목 추가:
```javascript
{ name: "굿즈명", price: "25,000원", emoji: "👕", category: "의류" }
```

## 선수 추가 방법

`PLAYERS` 배열에 새 객체를 복사해서 추가하면 됩니다.
`id`는 고유한 숫자여야 합니다.

## 디자인 커스터마이징

`index.html` 최상단 `:root` CSS 변수로 전체 색상 조정:
```css
--accent: #FF3300;    /* 주 강조색 */
--gold: #C9A23E;      /* 가격 표시색 */
--bg: #070A12;        /* 배경색 */
```

## 플랫폼명 변경

`LEGACY`를 원하는 플랫폼명으로 바꾸려면 HTML에서 `LEGACY`를 검색·교체하세요.
